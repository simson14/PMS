-----------------------------------------------------------------------------------------------------------------------
PMS : 개요
-----------------------------------------------------------------------------------------------------------------------
    1. SI 프로젝트 진행때마다 프로젝트 관리 및 커뮤니케이션을 위한 체계적인 관리체계(시스템)가 없는 관계로
       프로젝트 관리와 커뮤니케이션에 많은 비용과 시간이 소요 되므로 프로젝트 관리에 대한 비용과 시간을 줄이기 위하여
       Spring-boot 기반의 Web 사이트를 개발한다.

-----------------------------------------------------------------------------------------------------------------------
PMS : 목표
-----------------------------------------------------------------------------------------------------------------------
    1. 프로젝트 분석->설계->개발->테스트->상용적용 전단계의 각 Task들의 관리 및 추적
    2. 프로젝트 진행상황 공유
    3. 요구사항을 기반으로 관련 데이터, 액션, 액터들을 분석 할수 있도록 한다.
